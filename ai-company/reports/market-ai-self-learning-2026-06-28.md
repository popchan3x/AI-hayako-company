# 相場分析AI 自己学習システム 2026-06-28

## 結論
相場分析AIに、毎日シグナルを保存し、未来の価格で答え合わせし、モデル別の成績を積み上げる自己学習の土台を追加しました。

## 理由
最強の相場分析AIに近づくには、1回の分析精度よりも、毎日の予想、結果、反省を残し続ける仕組みが必要です。今日の実装では、全14対象について1日1回のシグナルを保存し、4本後、8本後、16本後の結果が確認できるようにしました。

## 実装内容
1. 日次学習コマンド
   - `npm run learn:daily`
   - 全14対象のシグナルを保存
   - 同じ日、同じ対象、同じデータ元の重複保存を防止

2. 学習データ保存先
   - `data/learning/signals.jsonl`
   - `data/learning/outcomes.jsonl`
   - `data/learning/learning-summary.json`
   - `data/learning/learning-summary.md`

3. 答え合わせ
   - 4本後
   - 8本後
   - 16本後
   - 方向、信頼度、主導モデル、相場環境、損益を保存

4. 集計
   - モデル別
   - 相場環境別
   - 対象別
   - 信頼度帯別

5. API
   - `GET /api/learn/summary`
   - `POST /api/learn/daily`

6. 画面
   - 累計シグナル
   - 答え合わせ件数
   - 未評価件数
   - 本日追加件数

7. 自動化
   - 毎日9:10に日次学習を実行する自動化を作成
   - 実売買、X直接投稿、外部決済、個人情報操作は0件

## 今日の結果

- 新規答え合わせ: 0件
- 未評価: 42件
