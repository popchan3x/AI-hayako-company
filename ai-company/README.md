# Hayako Market AI

短期デイトレ向けの相場分析AI Webアプリです。表示はすべて検証用で、実売買とX投稿はユーザーが行います。

## できること
- 米国株、主要指数、FX、XAUUSD、XAGUSD、XPTUSDを分析する
- 入口価格、損切り価格、利確価格つきの検証用シグナルを出す
- 5つのモデルを同じ条件で比べる「モデル大会」を行う
- 相場環境、データ品質、売買コスト、モデル一致度で信頼度を調整する
- 全91対象を一括で見る全体スキャンを行う
- 上向き、下向き、待機の3シナリオを出す
- 毎日のシグナルと答え合わせを保存して、モデル別の成績を積み上げる
- AI説明とX投稿用下書きを作る
- Xへ直接投稿しない

## 起動
```bash
npm start
```

ブラウザで `http://localhost:3000` を開きます。

## テスト
```bash
npm test
```

## 日次学習
```bash
npm run learn:daily
```

全91対象のシグナルを `data/learning/signals.jsonl` に保存します。未来の価格データがそろったシグナルは、4本後、8本後、16本後で答え合わせし、`data/learning/outcomes.jsonl` に保存します。

## 注意
このMVPは検証用です。無料データやデモデータには遅延、欠損、誤差の可能性があります。実売買の最終判断はユーザーが行ってください。
