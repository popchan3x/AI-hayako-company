# Hayako Market AI

短期デイトレ向けの相場分析AI WebアプリMVPです。

## できること
- 米国株、主要指数、FX、XAUUSD、XAGUSD、XPTUSDを分析する
- 価格付きの検証用シグナルを出す
- モデル大会で3つの基準モデルを比較する
- AI説明とX投稿用下書きを作る
- Xへ直接投稿しない

## 起動
```bash
npm start
```

ブラウザで `http://localhost:3000` を開きます。

## テスト
```bash
npm test
```

## 注意
このMVPは検証用です。
無料データやデモデータには遅延、欠損、誤差の可能性があります。
実売買の最終判断はユーザーが行ってください。
